# Get Stripe User Account

Get the stripe user account using the email as an identifier.

_API Secret Key_ - Specify your platform's Secret Key. Can contain [My Tokens](http://www.dnnsharp.com/dnn/modules/my-custom-tokens){:target="_blank"}.

_User email_ - The email of the connected account. Can contain [My Tokens](http://www.dnnsharp.com/dnn/modules/my-custom-tokens){:target="_blank"}.

_Account Number Token Name_ - Optionally provide a token name where to store the account number. Can contain [My Tokens](http://www.dnnsharp.com/dnn/modules/my-custom-tokens){:target="_blank"}.
